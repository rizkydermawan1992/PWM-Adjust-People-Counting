# PWM-Adjust-People-Counting

Description:
Control fan motor speed (PWM) based on number of people detected by camera

Software Required:
- IDE Arduino 1.8.3
- Python 3.8.5
- PIP 20.2.2
- OpenCV 4.3.0
- pySerial 3.4
- Numpy 1.19.1

Hardware Required:
- Laptop/PC
- Webcam
- Arduino Uno Board
- Cooling Fan 12V DC
- LCD I2C
- Jumper Wires
- USB Cable
